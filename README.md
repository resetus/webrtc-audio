# Тестовое приложение webrtc-audio
Приложение для записи голоса по технологии WebRTC

- **React**
- **NodeJS Express**
- **socket.io**  


### Запуск приложения
`yarn app` - запуск приложения по адресу http://localhost:3000

### Dev mode
`yarn start` - запуск *dev* (разработка) на webpack-dev-server по адресу http://localhost:3000

### Создание сборки
`yarn build` - запуск *prod* сборки приложения
